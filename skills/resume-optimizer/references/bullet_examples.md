## Common Weak Verbs to Avoid

### Replace "Responsible for" or "Worked on"
❌ Responsible for user authentication system
✅ Implemented OAuth 2.0 authentication system handling 500K daily logins with 99.9% uptime and sub-200ms response time

### Replace "Helped" or "Assisted"
❌ Helped improve application performance
✅ Optimized application performance by implementing database connection pooling and query caching, reducing average response time by 55% (from 1.2s to 540ms)

### Replace "Participated in" or "Involved in"
❌ Participated in Agile development process
✅ Drove sprint planning and retrospectives for 6-person team, increasing story completion rate from 70% to 92% and reducing sprint carryover by 60%

## Context Matters: Early Career vs Senior

### Junior Engineer (0-2 years)
Focus on: Technologies learned, projects completed, problems solved

Example: Developed RESTful API using Node.js and PostgreSQL for user management system, implementing JWT authentication and role-based access control for 1,000+ users

### Senior Engineer (7+ years)
Focus on: Business impact, technical leadership, system-level thinking

Example: Led architectural redesign of payment processing system handling $50M annual transactions, reducing infrastructure costs by $200K while improving processing speed by 3x and achieving PCI DSS compliance
